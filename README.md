# Expense Tracker (ASP.NET MVC)

A simple expense tracking web application built with ASP.NET Core MVC.

## Features

- Add expenses
- Edit expenses
- Delete expenses
- Category filtering
- Search expenses
- Expense totals
- Dynamic category dropdown

## Technologies

- ASP.NET Core MVC
- C#
- Razor Views
- LINQ
- Bootstrap
